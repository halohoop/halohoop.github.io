卫语句

在《阿里巴巴Java开发手册》.pdf中的15页有提到“卫语句”，如下图。

说的是：

我们不要这样写：

if(){

}else if(){

}else if(){

}...

而要这样写

if(){
    return;
}
if(){
    return;
}
if(){
    return;
}

虽然不是很明白更深层的原因，但是我表示对后面那一句“或者状态模式来实现”很有感触，因为正在做的一个小框架就打算使用状态模式来摒弃一堆的if-else判断。
