第七章 | 买入卖出我的第一只美股

| 本文为付费栏目文章，您已订阅，可阅读全文 |
好了，假设现在你已经在券商开了户，香港银行卡也办好了，钱也汇出到券商的账户上了，是时候可以小试身手，感受一下买卖美股的具体流程了。作为教程的最后一篇文章，我们就来说说对于第一次买卖美股的新手，交易过程中可能会出现的疑问。

美股的交易时间

中国在东八区，美国东部则是西五区，相差 13 个小时。同时，美国还有夏令时和冬令时之分，每年 3 月的第二个星期日到 11 月的第一个星期日为夏令时，时钟会拨快一小时，这时候中美的时差变为 12 个小时。因此，如果按中国的时间来看，美股的交易时间如下：
  • 夏令时：周一至周五的当晚 21:30 - 次日凌晨 4:00
  • 冬令时：周一至周五的当晚 22:30 - 次日凌晨 5:00
不过，美股市场除了正常交易的时间段外，还有盘前和盘后交易，就是说，这些时间段内你也可以买卖股票。同样，按照北京时间来看：
  • 盘前交易时间段：夏令时下午 16:00 - 晚上 21:30，冬令时下午 15:00 - 晚上 22:30；
  • 盘后交易时间段：夏令时凌晨 4:00 - 早上 8:00，冬令时凌晨 5:00 - 早上 9:00；
看到这里，你可能有些困惑，既然有盘前盘后交易时间，为什么不直接说美股的交易时间是下午 16:00 - 次日早上 8:00？为什么还要区分盘前盘后和正常交易时间？
你可以把它想像成你上班的工作时间。一般来说公司都会有统一的工作时间要求，例如早上 9:00 - 下午 17:00，那么这段时间内，理论上所有的同事都应该在公司上班，可以被找到。当然，除了公司要求的时间外，如果你的工作或项目特别忙，你当然可以提前上班，或者在晚上继续加班。
美股交易时间也有点类似，区分盘前盘后与正常交易时间段的最大标志，就是市场的流动性是否充足。在正常的交易时间段内，买方和卖方都上线来交易了，交易对手充足，成交快，价格波动连续。而在盘前盘后时间，参与交易人较少，流动性差，不容易成交，交易的价格波动也较大。
那么，为什么会有盘前盘后交易呢？实际上，最早的盘前盘后交易，是不面向个人投资者开放的。近年来，随着电子化交易的成熟，并且美股市场上除了美国本土外的境外投资者增多，这些众多的因素促成了盘前盘后交易面向个人投资者开放。
但一般来说，还是建议大家在正常时间段内交易。主要是前面提到的流动性问题，在盘前盘后交易，你很容易需要出更高的价格才能买到股票，或出更低的价格才能卖出股票。因此，在下单时，记得勾选「不允许」盘前盘后交易。

不过，例外情况也同样存在。例如，对于中概股而言，中国公司在白天曝光的丑闻，往往美股还没有开市，这时候进入盘前交易时间段，许多投资者就会开始抛售股票。或者，许多公司会选择在盘后发布财报等重大信息公告，超出市场的预期也会使得投资者在盘后买卖股票。

美股的交易限制

不限交易的最小单位

在「第一章 | 为什么选择投资美股？」中就提到过，美股交易不像中国的 A 股或港股，有最小交易单位的限制（如必须一次性交易 100 股），美股可以以 1 股为单位进行买卖。
不过，你仍然需要考虑交易成本的问题。在「答疑汇总 02」中提到过，当你算买美股的「人人」，当前股价是 9.44 美元。如果你一次性买入 100 股，手续费就算 2 美元（不同美股券商的收费可能不同），那么摊薄到每一股的买入成本是 (9.44*100+2)/100 = 9.46 美元。而如果你一次性只买入 1 股，那么这时候这一股的买入成本就高达 9.44 + 2 = 11.44 美元。

没有涨跌停限制

除了不限制最小交易单位，美股也不像 A 股有涨跌停的限制。A 股市场一只股票一天最多只能上涨或下跌 10%，在美股中则没有这个限制。一只股票可以一天下跌 50%，少掉一半的价格，也可以一天上涨 100%，股价直接翻倍。
不过，虽然美股对单日的涨跌幅没有限制，但出于对正常交易秩序的管理，会限制在极短的时间内价格发生剧烈的波动，这主要是通过对大盘股指实行熔断机制（Trading curb），以及对个股实行的限价机制（Limit Up Limit Down）来实现的。
对于大盘来说，熔断机制就相当于家庭中的保险丝，通过暂时切断电流（暂停交易），从而有时间查出剧烈波动的原因，或披露更多的信息，帮助投资者恢复理性，防止股市整体出现剧烈的波动。具体来说：
  • 在美国东部时间 9:30 - 15:25（即正常交易时间段结束前 35 分钟）内，如果标普 500 指数波动超过上一个交易日的 7% 和 13%，整个大盘会暂停交易 15 分钟，每天只会触发一次。
  • 在美国东部时间 9:30 - 16:00，如果标普 500 指数波动超过 20%，则当天股市暂停交易。
对于个股来说，则实行限价机制（Limit Up Limit Down），它的实际规则较复杂，根据交易时间、股票价格的不同，有较详细的规则，但笼统来说，它主要是限制一只股票在极短时间内价格发生剧烈的波动。如在正常交易时间段内，标普 500 指数的成份股，一旦股价波动超过前五分钟内股价的加权平均值的 5%，则暂停交易 15 秒钟，15 秒钟后若价格仍未回到正常区间，则暂停交易 5 分钟。

允许日内（T+0）交易

在 A 股市场中，你今天买入一只股票，必须要等到明天才能卖出。而在美股市场则不受这个限制，今天买进来的股票，今天就可以再卖掉。这被称为日内交易，也就是 T+0 交易。如果你的账户总资产在 25000 美元以上，则可以不受次数限制地进行日内交易；如果少于 25000 美元，则在 5 天之内只能进行 3 次日内交易，超出这个限制会被禁止交易 90 天。
一方面，这让日内波段交易成为了可能，另一方面，对于价值投资者而言，则有了及时纠错的能力。例如一只股票近期受负面消息影响股价一路下泄，而你认为负面消息并不属实，从而买入了股票。结果没过一个小时，负面消息被坐实了，这时候你可以及时再卖出这只股票止损。
需要注意的是，日内交易次数的判定，是以当日内头寸的变动方向计数的，这么说可能有些抽象，举例来说：
  • 当天买入 1 股，卖出 1 股：算 1 次日内交易；
  • 当天买入 10 股，先卖出 4 股，再卖出 6 股：算 1 次日内交易；
  • 当天买入 5 股，卖出 5 股，再买入 10 股，再卖出 10 股：算 2 次日内交易；
  • 当天买入 5 股，又买入 10 股，再卖出 10 股，再卖出 5 股：算 1 次日内交易；

美股的报价单

在买卖美股时，你会发现下单的时候，有不同的订单类型可以选择。事实上，美国的底层券商盈透提供了数十种不同的订单类型。不过，在国内美股券商 app 中，最常用到的往往只有 4 种订单类型。
在不同的券商 app 中，这些订单类型的名称也不尽相同。例如，雪盈证券中是「限价单、市价单、止损单、追踪止损单」，在富途牛牛中则叫做「限价单、市价单、条件单、跟踪止损单」。这么一看，你可能会觉得有点晕，而且每次切换不同的订单类型时，下单时需要填写的参数也不尽相同。

不过不用觉得头大，这些订单类型其实非常容易理解。从本质上来说，它们都符合 if - then 的结构，即当股票价格满足什么条件时，做出什么操作。
最基础的订单类型其实是「市价单」和「限价单」。它们的条件判断非常简单，只有一层 if - then 的结构。而高级类型的订单，如前面提到的止损单，往往是两级 if - then 结构的嵌套。

市价单

我们就先来说说市价单，它用 if - then 表示的话：
如果你是想买入股票
if 市场上存在卖一价
then 成交

如果你是想卖出股票
if 市场上存在买一价
then 成交
你可能会问，什么是「买一价」和「卖一价」呢？我们一起来看看一只股票的详情页，这里你可以看到三个价格：现价、买一价、卖一价。

股票价格实际上是买卖双方达成共识的结果。现价就是最后一次成交的价格，有人出 39.63 元愿意买入百胜中国，同时有人出 39.63 元愿意卖出百胜中国。在此时此刻，百胜中国正在等待着下一次买卖双方达成共识，来刷新股价。那么，最有可能成交的价格是什么呢？就是买一价和卖一价。
  • 买一价：离现价最接近、有人愿意买入的价格；
  • 卖一价：离现价最接近、有人愿意卖出的价格；
所谓「市价单」，其实就是去匹配市场上当前报价最接近现价的交易对手。它的目标，就是以最大的可能、最快的速度去成交，不过，需要相应承担的风险是，成交的价格不一定理想。
例如，一只股票正处在快速上涨、交易量抬升的阶段，这时候你如果想买入它，如果以「市价单」提交订单，在你提交的时候股价是 5 元，而在交易所收到并处理这笔订单时，股价已经上涨到了 6 元，并且最新的卖一价是 6.1 元，那么你最终的成交价格就是 6.1 元。

限价单

理解了什么是市价单，再来理解限价单就非常简单了。同样写成 if - then 的表示：
如果你是想买入股票
if 有人的卖出报价等于或低于你的买入报价
then 成交

如果你是想卖出股票
if 有人的买入报价等于或高于你的卖出报价
then 成交
简单来说，限价单是保障肯定以你满意甚至更优的价格来成交，但它不保证成交的时间，甚至如果等不到你心仪的价格，根本就不会成交。例如一只股票现在 5 元钱，你出 4.5 元的限价买入单，并不能马上成交，只有当有人出 4.5 元或更低的卖出报价时，才会摄合成交。

条件单

条件单其实是一个两层的 if - then 结构嵌套，它的本质还是一个限价单或市价单，只是增加了一个条件：即股票价格满足什么条件时，自动发出一个限价单或市价单。用 if - then 来表示的话：
if 股票价格达到一个特定的价格
then 发出一个「市价单」或「限价单」
例如，一个条件市价单：
if 股票价格达到一个特定的价格
then 发出一个「市价单」
{
如果你是想买入股票
if 市场上存在卖一价
then 成交

如果你是想卖出股票
if 市场上存在买一价
then 成交
}
同样的道理，条件限价单就是：
if 股票价格达到一个特定的价格
then 发出一个「市价单」
{
如果你是想买入股票
if 有人的卖出报价等于或低于你的买入报价
then 成交
如果你是想卖出股票
if 有人的买入报价等于或高于你的卖出报价
then 成交
}
在实际运用中，条件单最常见的运用场景就是止损或锁定利润。例如，你在 1300 元买入了亚马逊的股票，现在已经涨到了 1500 元。你在犹豫是继续持有还是抛掉，你决定提交一个条件单来锁定利润：如果亚马逊的股价下涨到 1450 元，那么就发出一个卖出市价单。这样一来，股价如果在 1500 美元的基础上继续上涨，你的条件单不会被执行，如果真的下跌到了 1450 元，你至少锁定了 150 元左右的利润。

追踪条件单

追踪条件单相比上述的条件单，在条件的设置上更复杂一些。条件单的条件，主要是股票达到了一个特定的价位，而追踪条件单允许你不设置某一个具体的价位，而是追踪股价的变动幅度，达到了一个特定的价差。
以最常见的追踪止损限价单为例，它的 if - then 结构是：
if 股票价格下跌幅度达到特定的价差
then 发出一个「卖出限价单」
{
if 有人的买入报价等于或高于你的卖出报价
then 成交
}
例如，还是前面的情况，你在 1300 元买入了亚马逊的股票，现在已经涨到了 1500 元。这时候你可以提交一个追踪条件单，当股价下跌幅度达到 50 元时，自动发出一个卖出限价单。如果亚马逊从 1500 元直接下跌了 50 元到 1450 元，那么就会生成这笔卖出限价单。或者，亚马逊又一路上涨，从 1500 元上升到了 1700 元，这时候下跌 50 元到 1650 元，才会生成这笔卖出限价单。
这就是美股下单中最常见的四种订单了，至此，你可以试着买入卖出你的第一只美股。

写在最后的话

至此，整个《价值投资美股新手指南》就告一段落了。不过，这可能只是你美股投资生涯的一个小小序曲，希望凭借着本教程，已经帮你厘清了基本的思路和操作。
然而，投资永远是一个动态的过程，没有谁完全准备好了，也没有谁能一直站在成功的顶峰。它本身就是一个「学习 - 实践 - 接受反馈 - 做出调整」的动态过程，正如我在这个系列教程中一直强调的，投资永远是一件科学与艺术相互交缠的领域，而科技和艺术，本身就没有尽头。
对于普通人而言，一来就想实践成长型的价值投资的确不是件容易事。对于你的第一只美股，建议你试着先拿少量的资金，在不动用杠杆（即不去融资融券，所谓杠杆，就是成倍地放大收益或亏损）的情况下，买入一些大盘股（如苹果、Facebook、阿里巴巴）尝试下。随着自己的进阶，慢慢地你会能够自己挑选出合适的投资标的。
